### 更新机构用户

+ 版本

  ```
  1.0
  ```

+ 作者

  ```
  木华黎
  ```

+ 修改记录

  ```
  木华黎   2015-12-14   初稿
  ```

+ 请求 url 地址

  ```
  PUT /api/users/{userId}/status
  ```

+ 请求参数示例

  ```json
  {
    "status": 1
  }
  ```

+ 请求参数说明

参数名 |  说明 | 必填 | 类型 | 长度 | 备注
-------|-------|------|------|------|-----
status | 用户状态 | 是 | 整形 | 2 | 0: 未激活, 1: 正常


+ 返回结果集
  - 正确结果示例, HTTP `STATUS=200`

    ```
    更新成功
    ```

  - 错误结果示例, HTTP `STATUS=500`

    ```
    更新失败
    ```
